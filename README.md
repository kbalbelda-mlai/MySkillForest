# My Skill Forest

## About

**My Skill Forest** is a more playful way of presenting a professional learning journey.

Instead of collecting every skill in a long list, this project turns them into a growing landscape. Broad areas of knowledge become **forest patches**, newly acquired skills begin as **saplings**, and skills supported by real evidence mature into full-grown **trees**.

This forest is only one part of a larger portfolio, but it is designed to grow alongside it. As more skills are learned, practiced, and applied, the forest becomes denser, more varied, and more alive.

## Take a Stroll

The latest public version of the forest is available here:

### [Visit My Skill Forest](https://kbalbelda-mlai.github.io/MySkillForest/)

The deployed site opens in **Strolling Mode**, where visitors can:

- move around the forest;
- open individual forest patches;
- inspect planted skills;
- view tree descriptions and dates;
- follow available evidence links; and
- search for specific trees.

The public version is read-only. Planting, editing, and deforestation are performed only in the local owner version.

---

## How the Forest Grows

Each tree represents one skill.

- A newly added skill begins as a **sapling**.
- Evidence is placed inside that tree's evidence folder.
- Once valid evidence is detected, the sapling grows into a **tree**.
- If all evidence is removed, the tree returns to sapling form.

The forest therefore reflects not only what I am learning, but what I can demonstrate through projects, notebooks, reports, certificates, analyses, and other supporting work.
